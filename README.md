# ai-marketing-wiki

Personal AI + marketing knowledge base.

This repo follows an llm-wiki style:

- `raw/` keeps original materials.
- `wiki/` keeps distilled, cross-linked notes maintained by AI.
- `_templates/` keeps Obsidian Web Clipper templates.
- `_attachments/` keeps images and files used by Obsidian notes.
- `workflows/` describes repeatable marketing analysis workflows.
- `prompts/` stores reusable prompts.
- `outputs/` stores generated briefs, reports, and analysis drafts.

The first goal is not automation. The first goal is to turn marketing knowledge and personal judgment into reusable assets.

Open this folder as an Obsidian vault to use it as a visual wiki.
