## 2. 邮件定位与点击目标

这封 EDM 的核心不是解释产品参数，而是借“决赛大战一触即发”的情绪，把用户推向 match-day 购买 / 颜色选择。

核心策略：

- Hero 用左右对阵和大面积撞色建立“最后一战”的紧张感。
- 过渡段用 `WITNESS HISTORY` 从观赛情绪转入购买动作。
- 产品模块用四个色块承接“四强”概念：每个板块一台风扇、一个 CTA 占位、一个可替换 team-color 方案。
- 设计必须可快速换色：四强未定时用 `Team A-D` 占位，四强确定后替换色块、产品颜色、CTA 语言和贴纸。
- 复盘时优先看 CTR、落地页点击、转化、RPE、退订；打开率只做辅助信号。

## 5. 视觉调性总要求

### 5.1 风格定义

中文执行名：**Final Showdown 撞色产品分区 EDM**。

关键词：

```text
final showdown / high-contrast color blocking / team-color inspired / split hero / product color panels / match-day tension / clean bold sports retail / reusable sticker accents
```

### 5.2 Diveblues 品牌要求

- 仍然要清凉、干净、明亮，不能做成燥热、脏乱的体育海报。
- 允许大面积高饱和撞色，但画面结构必须干净、边界清楚、移动端可读。
- 风扇是产品模块主角；Hero 中风扇也要有存在感，不只做装饰。
- 可以复用上一封的贴纸元素，但贴纸只做辅助层，不要盖过色块和产品。

### 5.3 颜色使用原则

- 可以使用球队颜色灵感。
- 不使用官方 palette，不复刻官方球队视觉系统。
- 不使用球衣、队徽、国旗、赛事图形。
- 基础模板用 `Team A / Team B / Team C / Team D` 占位，不写真实队名。

## 9. IP / 合规 / 设计避坑

### 可用

- 抽象 `final showdown` / `team-side` / `match-day` 表达。
- 球队颜色灵感与泛化撞色。
- 泛化球迷贴纸、足球、球票、风线、冰晶、产品 cutout。
- `Team A-D` 占位体系。

### 不使用

- FIFA / World Cup / Club World Cup / 联赛官方资产。
- 真实球队 logo、队徽、官方球衣、号码组合、官方 slogan。
- 国旗、赛事图形、官方 palette、国家队官方视觉系统。
- 真实球员 / 名人肖像。
- Stanley 商标、产品轮廓或原图结构的直接复刻。
- 暗示官方授权的表达，如 `official team color`、`licensed edition`。

### 设计避坑

- 不要把产品模块做成四张互相独立、难以统一替换的图；本需求要求一整张图。
- 不要把四个产品块做成真实球队联名页面。
- 不要使用球衣纹理或国旗图案来暗示队伍。
- 不要把撞色做得过脏、过暗、过热；Diveblues 仍要清凉、干净、明亮。
- 不要让 CTA 字太小，移动端必须能看清。
- 不要把所有文字压进图片而没有 alt / fallback。
