## 5. 整体视觉调性

### Diveblues 品牌要求

- 清凉、干净、明亮。
- 生活参与者视角：用户像在一个好玩的 match-day 场景里，而不是远远看产品摆拍。
- 动态：足球飞行、气流、门将动作、观众反应、贴纸箭头、交错模块都要有动势。
- 多色但不燥热：可以用草地绿、天空蓝、白、冰蓝、亮黄小面积点缀；避免大面积橙红、灰暗、脏乱。

### 本封视觉关键词

```text
playful / cool / sporty / bright / clickable / editorial sports illustration / magazine collage / real product cutout / match-day gear / team color
```

### 主推视觉风格

**Editorial Sports Illustration x Magazine Collage**

中文执行名：**体育编辑插画拼贴风**。

执行重点：

- Hero 不做动画片感，而是做一张有设计感的体育编辑插画主视觉。
- 产品尽量用真实产品图 / 产品 cutout，或半写实产品渲染，避免被画成玩具。
- 门将可以是简化插画人物，不追求五官写实，但动作和运动张力要清楚。
- 足球、气流、禁区线、箭头、贴纸、速度线用平面图形表达。
- 下方模块延续 Owala 参考图的纵向拼贴和手写标注，但整体更清爽、更 Diveblues。
- 推荐组合：**真实产品 + 插画场景 + 杂志拼贴排版**。

提示词中避免使用：

- `cartoon`
- `cute mascot`
- `children's illustration`
- `funny animated style`
- 任何受保护动画工作室名称

可使用：

- `editorial sports illustration`
- `mixed-media magazine collage`
- `real product cutout`
- `graphic airflow`
- `hand-drawn annotation`
- `clean vector-inspired illustration`
- `paper cutout texture`
- `premium youth sports editorial`

## 6. 模块视觉需求表

### 6.0 本次执行统一风格

本次第 6 节和第 8 节为最新执行口径，视觉上优先使用 **Americana Sports Paper-Cut Sticker Collage**：

- 中文执行名：**美国体育文化剪纸贴纸拼贴风**。
- 主色：**红色 / 米白色 / 海军蓝色（深蓝）**，作为本稿指定的美国世界杯视觉代表色。
- 撞色方式：红色负责冲击点，米白负责纸张底色和呼吸感，海军蓝负责结构和高级感；可用少量白色和旧纸纹理，但不回到大面积浅蓝 / 草绿。
- 美国文化元素：stadium lights、ticket stub、tailgate parking lot、folding chair、cooler、varsity number shapes、generic pennants、bumper sticker、sports program paper texture。
- 视觉密度：少文字、少参数、少堆叠；用剪纸层次、贴纸边缘、图形风线和留白制造美感。
- 产品呈现：优先真实产品 cutout / 半写实产品渲染，避免产品被画成玩具。
- 禁止：官方 FIFA / World Cup 资产、官方球队队徽、真实球员肖像、官方球衣、国旗直接复刻、密集电商 banner、红金促销感。

## 9. IP / 合规 / 设计避坑

### 必须避免

- FIFA / World Cup 官方 logo、奖杯、官方视觉系统。
- 官方球队队徽、国家队球衣、真实队服版型、球衣号码。
- 真实球员肖像、明星球员动作过度还原、姓名或昵称。
- 官方赛事转播画面、比分、赛程事实，除非另行核验和授权。
- 受保护动画工作室风格的直接点名或复制。
- 未确认的折扣、赠品、免邮、抽奖、surprise。
- 未确认性能比较：如 `cools faster`, `more air than...`, `strongest`, `guaranteed`。

### 设计上避免

- 首屏只看到产品，没看到故事。
- 长图中段变成参数表。
- CTA 过多，导致点击目标不清。
- 大面积红橙热感压过 Diveblues 的清凉明亮。
- 拼贴过密，移动端无法读。
- 使用真实球队 / 官方赛事元素来制造“世界杯感”。

### 推荐做法

- 用足球场、球门、足球、禁区线、风线、观众色块建立泛化足球语境。
- 用 `match day`, `goal`, `clean sheet`, `team color` 等语言做创意表达。
- 用清爽蓝白底、足球贴纸、手写箭头、图形速度线做年轻互动感。
- 用产品参与故事，而不是产品独自摆拍。
