- 不含：产品规格与卖点（产品部负责）；红人日历、联盟日历（本方案整体轮空，仅保留角色定位）

## 0. 这份方案要解决什么

让运营、视觉、品牌三方对齐到同一套理念：2026 冬季项目的核心主题 + 怎么打入市场。 内容核心是"实际打法"，概念层从简、打法与日历从详。

## 1. 执行摘要

- 核心主题：筑底（Go）打未来（Sense）；品牌内核：亲民务实，时尚生活。
    
- 一句话打法：社媒先行做"认知 + 人设 + 话题"，Go 跑量打底、Sense 立旗打未来；IG 主阵地、TK 话题引擎、FB 礼赠与广触达。
    
- 创新主线（老树开花）：一套统一的 O 圆环系统 —— 所有产品加挂环，O 环元素进 logo + 新包装 + 新 APP，把产品/品牌/数字生态串成一个符号。
    
- 节奏：产品 8 月中旬上市（APP 开售即就绪）→ 9 月–次年 2 月为社媒主战役期。
    
- 趋势底气：短视频是互动主引擎、真实内容跑赢种草、社交电商礼赠渗透高、cozy/under-consumption 生活方式成势、智能数据型 wellness 产品高接受度（数据见第 3 节，附来源）。

## 2. 核心主题与品牌定位

### 2.1 核心思想解码

|   |   |   |   |
|---|---|---|---|
|新品|角色|品牌内核|战略任务|
|Go（基础款）|筑底|亲民务实|跑量、铺渗透、降认知门槛、做"第一支随身暖物"|
|Sense（智能带定位旗舰）|打未来|务实、生活|立创新旗帜、拉高品牌天花板、抢占"未来/智能/生态"心智|

一句话：Go 守住基本盘的量与口碑，Sense 把品牌往"时尚 + 智能 + 生态"推。 两者是"地基 + 楼"的关系，不是两个孤立 campaign。

### 2.2 产品定位细化

Go · 亲民务实 · 筑底

- 心智占位：可靠、好懂、随身。
    
- Slogan：Grab. Loop. Go.
    
- 角色：拉新主力 + 转化基本盘 + UGC 量产载体。
    
- 情绪：务实、安心、日常、无负担。
    
- 人群入口：EDC/gear、通勤、户外轻度、礼赠（性价比）。
    

Sense · 时尚生活 · 打未来

- 心智占位：智能、定位、APP、懂你温度、可被生活方式认领。
    
- Slogan：Smart. Found. Personal.
    
- 角色：品牌天花板 + 旗舰声量 + O 环生态与 APP 的锋利端。
    
- 情绪：时尚、向往、soft wellness、被理解。
    
- 人群入口：soft wellness/body-state、lifestyle/gifting（高端）、科技尝鲜。
    

### 2.3 老树开花：O 圆环

创新不是开新品类，而是一套统一的"O 圆环"：

- 所有产品新增挂环（O-ring） —— 从"握在手里"升级为"挂得住、随身带"的 EDC 物件；
    
- O 圆环元素同时进入 logo + 即将升级的包装 + 即将上线的 APP —— 产品识别、品牌识别、数字生态共用一个符号。
    

落实"三位一体"纪律：产品(挂环) + 视觉(O 环 logo/包装) + 内容/文案 讲同一个故事

## 4. 市场进入打法（社媒方向）

### 4.1 节奏

8 月中上市（预热与首发在窗口前完成）→ 9–10 月铺认知与场景 → 11 月（夏令时后变冷 + BFCM）双峰：Sense 旗舰强推 + 全线转化 → 12 月礼赠 → 1–2 月深冬 soft wellness + 户外 + 情人节收尾。

### 4.2 Go 打法细化（筑底 · 亲民务实）

- 对标模式：Frost Buddy（真实问题 → 演示 → 打消顾虑）。
    
- 内容形式：短平快可复制、UGC 量产、测评/对比。
    
- 平台：TK 做测评/话题/趋势参与（高互动）；IG Reels 做场景；FB 触达更广年龄层 + 性价比礼赠。
    

### 4.3 Sense 打法细化（打未来 · 时尚生活）

- 内容支柱：Value Reframing（舒适物 + soft wellness + 时尚 EDC）+ O 环生态故事（产品→logo→包装→APP）。
    
- 内容形式：旗舰上市、APP+定位+数据解读短视频、O 环生态 explainer、aspirational 生活方式视觉、"where you left it"定位演示。
    
- 平台：IG 主打（人设/审美/Reels reach）；TK 做 persona series + 文化层；FB 适配高端礼赠。

## 5. 内容形式

Instagram（主阵地）

- Reels（主力）：felt-cold POV 微剧情；tactile ASMR（按键/升温质感）；before→after relief；Sense 定位/APP 演示；O 环挂带穿搭。
    
- Carousel：gift guide、O 环系统科普、人群场景合集。
    
- Stories：投票/问答、幕后、限时礼赠提醒。
    
- Grid 审美：Sense 高端时尚视觉锚点。
    

TikTok（话题/测试引擎）

- Persona series："the friend who knows how to live comfortably"。
    
- 反复测试系列（Frost Buddy 式）、评论区回应式、trend/sound 参与。
    
- Sense 数据解读（Oura 式 signal→interpretation）。
    

Facebook（礼赠/广触达）

- Gift guide、较长文案、链接到 Shop、面向年长送礼人群、社群互动。
    

通用节日形式：礼盒/O 环包装开箱、last-minute 礼物清单、BFCM 限时。

## 7. 跨部门分工（冬季项目）

- 社媒（本方案详化）：品牌人设、情绪与话题的窗口 + 流量引擎。IG 主阵地。详见第 4–6 节。
    
- 红人 / KOL（日历轮空）：放大社媒主题、用真实场景做产品可信度与示范、按四类人群分别触达；Go 侧重量化铺测评、Sense 侧重质化种草。本方案不排红人日历。
    
- 联盟 / Affiliate（日历轮空）：承接转化、节日礼赠导购、在 BFCM/圣诞等节点放大成交。本方案不排联盟日历。
