            Regression Analysis
Importance of regression analysis
• Regression analysis is one of the most important methods in quantitative market research
• Marketing and strategy departments of major companies (e.g. , Procter & Gamble, BMW Group, Nestlé)
  use regression analysis regularly for decision-making


Regression analysis can
    1. Indicate if the independent variables have a significant relationship with the dependent variable.
    2. Indicate the relative strength of different independent variables’ effects on a dependent variable.
    3. Make predictions.



* In this lecture, we focus on linear regression (OLS)

The Regression Analysis Estimate a Line Using
the Observations

                      The regression estimation procedure
                      attempts to fit a best possible line. This line
                      is optimal if the squared distances to all
                      observation points are minimized. Thereby:
                      1.   Positive and negative deviances don’t
                           balance each other
                      2.   Bigger deviances are weighted
                           disproportionately high
                      If the assumptions of the regression are
                      harmed, the line might not be the true line!

          Why do we need an Adjusted R2?
Properties of R2
• Measures the strength of association (or predictive power) in a multiple regression
• It usually increases as more independent variables are included


Adjusted-R2
• Adjusts for the number of independent variables and sample size to account for
  diminishing returns → penalizing too many independent variables (overfitting)


In practice, we choose the best model based on both R 2 and Adjusted-R2

             Collinearity/Multi-collinearity
Collinearity occurs when two or more predictors in a regression model are highly correlated.
Why It Matters
• Inflates standard errors → unstable coefficient estimates.
• Undermines the interpretability of individual predictor effects.
Detecting Collinearity
• Variance Inflation Factor (VIF) > 10 (or VIF > 5 for small sample) signals a problem.
Example
• Marketing spend and promotional discount rates in the retail industry often rise together, making it hard
  to isolate each effect on sales.
Mitigation Strategies
• Remove or combine correlated variables.
• Use principal component analysis (PCA) to create uncorrelated components.                                   25

            1. Check the regression analysis data
            requirements
Sample size
• Formal power analysis (e.g. using G*power)
Enough variation in your data
Scale type of the dependent variable: interval or ratio scaled
Collinearity
• Collinearity is a data issue that arises if two independent variables are highly correlated.
• The VIF, or Variance Inflation Factor, can indicate collinearity. Values over 10 indicate collinearity.
• If collinearity is present, either use factor analysis, re-specify the regression model, or simply
  accept the collinearity (noting its presence).
    1.   variable elimination: remove one of the highly correlated variable; or
    2.   variable reduction: combine the highly correlated variables into one variable, or factor
         analysis

            2. Specify and estimate the regression model
The goal is to build a simple yet complete model (e.g., SPSS).
Some suggestions:
• If independent variables have overlap, then focus on the most important or distinct variable.
• If you need to use a model for different circumstances, ensure the independent variables are the same to
  allow comparison.
• Consider the type of advice you want to give.
• Consider the sample sizes and rules of thumb. Smaller datasets require fewer independent variables
• If independent variables are nominal/ordinal, try to use dummy variables.




                                                                                                         30
            3. Test the regression analysis assumptions
Critical assumptions
1. The regression model can be expressed linearly,
2. The regression model’s expected mean error is zero,
3. The errors’ variance is constant (homoscedasticity), and
4. The errors are independent (e.g., no autocorrelation).


Optional assumption (to determine the significance of parameters)
5. The errors need to be approximately normally distributed.

             4. Interpret the regression results
Effects of Individual Variables
1. The null and alternative hypotheses for each coefficient:
  • H0 : b i = 0
  • H1 : b i ≠ 0
  • If the Sig. column in SPSS indicates a value below .05, we consider this to be significant.
2. If the coefficient is significantly different from zero, report:
  • The direction (+ / -)
  • The magnitude of the coefficients and the relative importance.
  – Unstandardized (B): expresses the effect of a one-unit change in the independent variable on the dependent variable.
  – The standardized coefficient (Beta): the effect relative to the other variables. Standardized effects can be compared but
    not for dummy (binary) variables

            Nominal independent variables
• DV of regression should be interval/ratio scaled. How about the IVs?
• Sometimes researchers are interested in categorical IV (e.g., sex, region, traveler status)
 • We can generate dummy (binary) variables coded as 0/1 and include the dummy variables as the
    independent variable.
 • However, when you include dummy variables, you must choose one category as the reference
    category.
Example:
   We want to include gender (female/male) as a predictor in our regression model. However, the unit of
   gender has no meaning. Hence, we create a dummy variable gender_dummy in the following fashion:
    o gender_dummy = 0 if gender = female;
    o gender_dummy = 1 if gender = male.

    Then we include gender_dummy into the regression model.
