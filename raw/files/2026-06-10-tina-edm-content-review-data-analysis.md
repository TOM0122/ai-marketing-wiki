# 二、EDM 数据整体表现分析

## 1. 打开率分析（Open Rate）

### 表现最好的内容：

| 类型    | 打开率特点              |
| ----- | ------------------ |
| 节日送礼型 | 最高（圣诞 41%、黑五 38%+） |
| 强折扣型  | 稳定高打开              |
| 情绪型标题 | 明显高于纯功能型           |

### 当前发现：

* “Holiday / Black Friday / Gift” 类关键词天然具备更高吸引力
* 单纯功能表达（如 Daily Commute）打开率相对一般
* “Summer / Warmth / Cool” 情绪词对打开率有明显帮助

### 方法论：

> 用户首先打开的不是产品，而是“情绪”与“节日氛围”。

---

# 2. CTR（点击率）分析

## CTR 表现最高内容：

| EDM        | CTR    |
| ---------- | ------ |
| 黑五（11月20日） | 2.56%  |
| 黑五（11月18日） | 2.47%  |
| 早鸟活动       | 2.205% |
| 秋季大促       | 1.84%  |
| 盲盒         | 1.7%   |

---

## CTR 核心规律：

### （1）强折扣明显提升 CTR

例如：

* Up to 50% OFF
* Black Friday
* Early Bird

都属于：

# 强行动驱动型内容

用户点击动机明确。

---

### （2）场景型内容 CTR 普遍偏低

例如：

* 通勤
* 出游
* 音乐节

CTR 普遍在：
0.5%-0.8%

说明：

> 用户会觉得“好看”，但未必立即购买。

因此：

# 场景型更适合：

* 品牌建设
* 用户教育
* 提升品牌感知

而不是直接冲销量。

---

# 三、品牌内容方法论总结（重点）

# OCOOPA

## 用户核心购买逻辑：

### 1. 节日送礼

表现最稳定：

* 黑五
* 圣诞
* 母亲节

---

### 2. 强折扣刺激

CTR 和销售额明显更高。

---

### 3. 女性/家庭情绪视觉有效

适合：

* 礼物
* Holiday
* Warmth

---

### 4. 纯功能内容偏弱

用户并不太关心参数本身。

更在意：

# “温暖感”

而不是：
“技术感”

---

# Diveblues

## 用户核心购买逻辑：

### 1. Lifestyle 场景 > 参数

例如：

* 出游
* 音乐节
* 夏季
* 通勤

---

### 2. 情绪型标题效果更好

例如：

* Summer is Coming
* Stay Cool

比：

* High Speed Fan

更适合。

---

### 3. 男性生活方式场景潜力较大

适合：

* 体育赛事
* 世界杯
* 户外
* 通勤
* 演唱会
