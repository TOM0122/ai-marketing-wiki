# 目录

# 1. 工作概述

这份文档用于交接 Diveblues 官方 Instagram 账号的「红人建联」日常事务。你的核心任务是：在主动私信我们的 KOL / KOC / UGC 小达人中筛选合适人选，与他们沟通并促成内容合作，最终为品牌沉淀社媒内容。

整个工作可以拆成两类合作模式：免费产品置换合作，以及付费合作。两类合作的流程不同，但有一个共同的硬性要求——所有合作，不论免费还是付费，都必须完整记录在 Excel 表格里。

|  |
| --- |
| **一句话记住核心目标**  在「主动来私信我们的人」里找到合适的红人，谈成合作，把产品发出去，并且全程在 Excel 里跟踪到对方产出内容为止。 |

## 1.1 两种合作模式速览

| **对比项** | **免费产品置换合作** | **付费合作 (Paid Collaboration)** |
| --- | --- | --- |
| **核心做法** | 免费寄送产品，换取对方自发产出内容 | 支付费用，换取约定好的内容产出 |
| **对内容质量** | 一般不强求，对方不跑单即可 | 需审查数据与内容风格，质量有要求 |
| **主要价值** | 沉淀 UGC 内容 | 获得更可控、更优质的合作内容 |
| **关键动作** | 问地址 → 寄产品 → 记录 | 审数据 → 砍价 → 谈脚本 → 寄产品 → 记录 |
| **是否需要汇报** | 无需特别汇报，按流程执行即可 | 谈到合适人选与价格后，需向我汇报确认 |

# 2. 每日操作入口：找到主动私信我们的人

每天的工作从查看 Instagram 后台私信开始。请严格按以下步骤进入正确的消息分类。

1. 登录 Diveblues 的官方 Instagram 账号（使用网页版操作更方便）。
2. 点击 Message（顶部的小飞机 ✈ 图标），进入私信界面。
3. 网页版的私信界面会有三个分类标签：Primary、General 和 Requests。
4. 点击 Requests 标签。这里会看到大量主动后台私信我们的 KOL、KOC 或 UGC 小达人。
5. 你的建联工作，就是在这些「主动来找我们」的人当中筛选并展开合作。

|  |
| --- |
| **重要：首次回复时必须分类**  和 Requests 里的陌生人第一次发消息时，IG 会要求你决定把这个对话归到哪个分类。请一律默认归到 General，方便后续统一管理和查找。不要留在 Requests 里。 |

## 2.1 三个私信分类的含义

* Primary：主要联系人对话，通常是更重要或已建立关系的往来。
* General：一般对话。我们所有建联中的红人，首次回复后都归到这里。
* Requests：陌生人主动私信的待处理请求池，是你每天筛选的「来源」。

# 3. 完整建联流程总览

不论免费还是付费，整体流程都遵循同一条主线，只是在「谈合作」这一步之后分成两个分支。请先记住这条主线：

|  |
| --- |
| **流程主线**  筛选 → 建联 → 谈合作（免费 or 付费）→ 〔分支〕→ 记录 → 催产出（保持建联）→ 保存产出 |

## 3.1 免费合作分支

筛选 → 建联 → 谈合作 → 直接发货 → 记录 → 催产出（保持建联）→ 保存产出

## 3.2 付费合作分支

筛选 → 建联 → 谈合作 → 谈金额、谈脚本主题 → 发货 → 记录 → 催产出（保持建联）→ 保存产出

*两个分支的差别只在中间一段：免费合作确认地址后直接发货；付费合作需要先谈拢价格、再谈清楚脚本主题，然后才发货。其余环节完全一致。*

# 4. 分步操作详解

## 第 1 步 · 筛选

进入 Requests 后，对主动私信我们的人逐个做初步判断。在发消息之前，先大致看一下对方的主页。

* 查看对方的主页数据：粉丝量、互动情况、近期内容表现。
* 查看对方的内容风格：是否与 Diveblues 的品牌调性、产品类目相符。
* 初步判断对方属于哪一类——值得做免费置换的 UGC 小达人，还是有付费合作潜力的红人。

## 第 2 步 · 建联

对筛选后觉得合适的人，开始发私信建立联系。

* **首次发消息时记得分类：**把该对话默认归到 General。
* 结合你看到的主页数据和风格，发出第一条沟通消息。
* **核心要问清楚一件事：**对方想要的是免费的产品置换合作，还是 paid collaboration（付费合作）。这个答案决定了接下来走哪个分支。

## 第 3 步 · 谈合作（分支判断点）

根据对方的回答，进入对应分支：

* 对方接受免费产品置换 → 走「免费合作分支」（见第 4-A 步）。
* 对方要求付费合作 → 走「付费合作分支」（见第 4-B 步）。

## 第 4-A 步 · 免费合作分支

免费合作流程比较简单直接：

1. 直接询问对方的收件地址。
2. 安排寄送产品。
3. 把这笔合作完整记录进 Excel（见第 5 章）。

|  |
| --- |
| **关于免费合作的质量预期**  免费置换合作一般不去强求内容质量，对方只要不跑单（收了产品后正常产出内容）就很 OK 了。这类合作的主要目的是为品牌沉淀 UGC 内容。 |

## 第 4-B 步 · 付费合作分支

付费合作需要更谨慎，核心是「审查 + 砍价 + 谈脚本」这一套动作。

* **审查：**仔细核对对方的账号数据和内容风格，确认值得付费投放。
* **砍价：**放心大胆地砍价。对方的报价通常虚高，目标单价是 100–200 美元，其中 100 美元的情况占多数。
* **谈脚本主题：**价格谈拢后，与对方确认内容脚本和主题方向。
* **汇报：**找到合适人选并且砍到合适价格后，先跟我说一声、确认后再推进。

1. 谈金额（审数据 + 砍价至 100–200 美元区间）。
2. 向我汇报合适的人选与价格，确认。
3. 谈脚本主题，确认内容方向。
4. 安排寄送产品。
5. 把这笔合作完整记录进 Excel（见第 5 章）。

|  |
| --- |
| **预算提醒**  当前社媒预算有限，付费合作的平均单价控制在 100–200 美元，多数应落在 100 美元左右。对方报价虚高是常态，不用不好意思，大胆砍。 |

## 第 5 步 · 记录（两个分支都必须做）

无论免费还是付费，每一笔合作都必须记录到 Excel 表格里。这一步不可跳过，详细字段见第 5 章。

## 第 6 步 · 催产出（保持建联）

产品寄出后，红人的产出周期大约在 2 周到 1 个月。在此期间需要保持联系、适时催促对方产出内容。

* 依据 Excel 记录跟进每个人的进度，避免遗忘或搞混。
* 保持友好沟通，既是催产出，也是维护长期关系。

## 第 7 步 · 保存产出

对方产出内容后，及时保存好相关素材，并在 Excel 中更新状态，标记该笔合作已完成产出。

# 5. Excel 记录规范（重中之重）

|  |
| --- |
| **为什么必须记录**  第一，方便审查对方是否真的产出内容、是否逃单；第二，方便自己管理合作进度、催促产出。红人产出周期长达 2 周到 1 个月，不记录在 Excel 里非常容易遗忘和搞混。 |

每一笔合作（免费或付费）在确认后都要立即在 Excel 中建一行。建议至少包含以下字段：

| **建议字段** | **说明** |
| --- | --- |
| **红人账号 / IG handle** | 对方的 Instagram 用户名，便于查找对话。 |
| **主页数据** | 粉丝量、互动等关键数据快照。 |
| **合作类型** | 免费置换 或 付费合作。 |
| **合作金额** | 付费合作填最终谈定的价格；免费填「免费」。 |
| **收件地址** | 寄送产品所用的地址。 |
| **发货日期 / 状态** | 产品是否已寄出、寄出日期。 |
| **脚本 / 主题** | 付费合作约定的内容方向（免费可留空）。 |
| **产出截止 / 预计时间** | 按 2 周至 1 个月周期估算的产出时间。 |
| **产出状态** | 未产出 / 已催促 / 已产出 / 疑似逃单。 |
| **产出内容链接** | 对方产出后保存的内容链接或素材位置。 |
| **备注** | 沟通中的特殊情况、需要跟进的事项。 |

*注：以上字段为建议项，可按现有表格结构调整，但「合作类型、金额、发货状态、产出状态」属于核心字段，不能缺。*

# 6. 关键注意事项与每日清单

## 6.1 必记的几条规则

* 只在 Requests 分类里找主动私信我们的人来建联。
* 首次回复陌生人时，把对话默认归类到 General。
* 发消息前一定先看对方主页的数据和风格。
* 务必先问清楚：免费置换还是付费合作。
* 付费合作要审数据、砍价（目标 100–200 美元，多为 100），并先向我汇报确认。
* 不论哪种合作，全部都要记进 Excel——没有例外。
* 产品寄出后持续跟进，产出周期为 2 周至 1 个月。

## 6.2 每日工作清单

1. 登录官方 IG，打开 Message → Requests。
2. 逐个筛选新的主动私信，判断是否值得建联。
3. 对合适人选发起首次沟通，并把对话归类到 General。
4. 问清合作意向（免费 / 付费），进入对应分支推进。
5. 付费合作谈到合适价格后向我汇报。
6. 所有已确认的合作及时记入 Excel。
7. 回看 Excel，催促到期未产出的红人。
8. 收到产出内容后保存素材并更新表格状态。

|  |
| --- |
| **有疑问随时问**  遇到拿不准的情况——比如对方报价远超预算、内容风格不确定是否合适、或对方疑似逃单——不要自己硬扛，直接来问我。 |
