# Diveblues Go 冬季视觉与内容策略

> 来源：`raw/files/暖手宝品牌拓展视觉策略.pdf` 与其转写版 `raw/files/暖手宝品牌拓展视觉策略.md`。
> 定位：Diveblues Go 冬季视觉策略 / 内容策略上游口径，用于对齐包装、社媒、红人、联盟和品牌内容表达。

## 来源与定位

这份源文件原本以“暖手宝品牌拓展视觉策略”为题，讨论的是新品牌 / 新产品线在暖手宝冬季市场中的视觉进入方式。结合当前策略调整，Anbooc 不再独立展开，Go 系列完整转移到 Diveblues，因此这份文件应被理解为 Diveblues Go 冬季视觉与内容策略的核心参考。

它的价值不在于提供具体执行日历，而在于给出三个上游判断：

- 暖手宝市场正在从单一功能需求，转向情绪价值、礼品化、可爱化、图案化和更精细的人群偏好。
- Diveblues Go 需要和 Ocoopa 拉开生态位，不能只是 Ocoopa 的低价版本。
- “温暖”不一定只能用橙色、红色等传统暖色表达，可以通过触感、情绪、舒适感和人文关怀建立温暖认知。

## 一句话结论

Diveblues Go 冬季视觉不应复制 Ocoopa 的温情旗舰感，也不应落入传统暖手宝的橙红暖色同质化；它应以年轻、时尚、好玩、明快的方式，把 under-$15 暖手宝表达成一种能被随手携带、随手送出、随手分享的 winter EDC。

可转译为执行口径：

```text
Diveblues Go = under-$15 winter EDC, not a cheaper Ocoopa.
```

## 核心视觉策略

源文件给出的视觉关键词是：

```text
年轻 / 热情 / 时尚 / 好玩 / 明快
```

对 Diveblues Go 的含义是：

- **年轻**：面向学生、年轻通勤、EDC 用户、预算礼物人群，而不是成熟家庭礼赠主叙事。
- **热情**：可以有情绪张力和即时吸引力，但不等于大面积橙红暖色。
- **时尚**：产品需要进入包、穿搭、桌面、校园、通勤等日常生活搭配。
- **好玩**：允许 sticker、doodle、meme、夸张文字和轻互动，而不是纯参数说明。
- **明快**：视觉上要快速抓住注意力，尤其在包装、社媒首帧和联盟图中建立记忆点。

这与 Diveblues 既有“生活的参与者”“轻松、自由、快乐”“真实、多变、多面、动态”是兼容的。区别在于：夏季 Diveblues 主要用清凉、干净、明亮建立风扇心智；冬季 Diveblues Go 可以在保持明快和动态的同时，加入更年轻、更街头、更 sticker 化的社媒表达。

### 视觉 moodboard 线索

PDF 原件中的视觉 moodboard 不只包含文字，还包含一组明确参考：

- 宠物、动物、meme 感图像：支撑“可爱化”“好玩”“社媒传播感”。
- 时尚穿搭与高饱和彩色小物：支撑“时尚”“年轻”“随身搭配”。
- 雪地、滑雪、户外冬季画面：支撑“冬季场景”，但不是传统厚重暖手宝视觉。
- 彩色图形包装、字体包装、强符号包装：支撑“夺人眼球”“强化品牌记忆点”。
- 冷环境 + 舒适状态的组合：支撑“温暖不等于暖色”，可以用场景和人物状态建立温暖感。

因此，Diveblues Go 的视觉不是“暖手宝产品图 + 橙红色背景”，而应更像一个年轻化生活方式小物的 moodboard：可爱、有梗、时尚、明快，同时能落入真实冬季场景。

## 色彩策略

源文件最重要的判断是：

```text
表达温暖，并不需要框死在暖色系上。
```

这对 Diveblues Go 很关键。传统暖手宝视觉容易依赖橙色、红色、火焰、暖光等直接符号，但这些符号已经趋向同质化。Diveblues Go 更适合用一套组合逻辑表达温暖：

- 触感：手、口袋、包挂、握持、桌面随手拿起。
- 情绪：冷天的小烦恼被轻松解决。
- 感知舒适：不用煽情，也不用强调痛苦，只表达“刚好舒服”。
- 人文关怀：小礼物、朋友、室友、同学、通勤搭子。
- 年轻语气：用 meme、贴纸、短句和真实小场景建立共鸣。

因此，Diveblues Go 的冬季视觉可以跳出传统暖色：

- 可以使用更高饱和、更明快、更有对比的配色。
- 可以使用冷暖对比来表现“外部环境冷，产品带来舒适”。
- 可以延续 Diveblues 的清爽感，但不要让画面显得冰冷或缺乏温度。
- 暂不把“多颜色矩阵”作为产品核心卖点，因为当前 Go 实物可能与多颜色不强相关。

## 包装策略

源文件强调包装要：

- 遵循品牌视觉逻辑，保持品牌语言一致性。
- 在起势阶段快速占领消费者心智。
- 夺人眼球。
- 强化色彩记忆点和品牌记忆点。
- 相对削弱产品本身的表达。

对 Diveblues Go 来说，包装不应只承担“说明这是暖手宝”的功能，而应承担“这是 Diveblues 的年轻冬季小物”的识别功能。

包装方向建议：

- 主视觉优先传递年轻、明快、好玩，而不是传统暖手宝科技参数感。
- 包装短句要轻，例如 `Catch the Warmth`、`Cold Days. Good Vibes.`、`Your under-$15 winter EDC.`。
- 可使用 sticker / doodle / icon / 小标签语言，强化社媒感。
- 功能卖点放在辅助位置，避免包装正面被参数占满。
- 如果要体现温暖，不必只依赖橙红色；可以用触感场景、人物状态和冷暖对比完成。

## 内容策略转译

源文件中的市场判断可以转成以下内容策略：

| 源文件判断 | Diveblues Go 内容转译 |
|---|---|
| 功能性需求转向情绪价值需求 | 内容不能只讲快热、续航、价格，也要讲冷天真实小情绪 |
| 礼品化、可爱化、图案化出现 | 强化 stocking stuffer、under-$15 gift、可爱小物、包装感 |
| 购物偏好从模糊一致转向细分精准 | 内容要分 campus、通勤、EDC、礼物、轻户外，而不是一套素材打全部人 |
| 市场成熟带来下沉性价比和年轻化潮流化 | Diveblues Go 应做年轻、好玩、性价比，不做 Ocoopa 低配 |
| 新品牌要和 Ocoopa 拉开生态位 | Ocoopa 讲旗舰与信任，Diveblues Go 讲轻决策与年轻冬季小物 |
| 温暖不必框死在暖色系 | 视觉可以用明快色彩、冷暖对比、触感和情绪表达温暖 |

## Diveblues Go 的 EDC / seasonal carry 表达

当前双品牌方案已经将 Diveblues Go 定义为：

```text
under-$15 winter EDC
```

这份视觉策略进一步支持这个方向。因为它强调年轻化、性价比、潮流化、好玩和包装记忆点，而 EDC 正好可以把这些元素连接起来。

建议稳定使用的表达：

- winter EDC
- everyday carry warmth
- seasonal carry
- under-$15 winter gear
- campus carry
- commuter carry
- small gift that actually gets used

视觉上，EDC 不应拍成硬核工具清单，而应拍成年轻人的日常小物组合：

- 包、钥匙扣、耳机、唇膏、学生证、交通卡、热饮、Go。
- 书包 / 通勤包 / 外套口袋 / 桌面 / 图书馆 / 地铁站。
- sticker 标签标出每个小物的角色。
- 画面中可以出现“冷天但不狼狈”的状态，而不是痛苦受冻。

## 社媒可用角度

### 1. Winter EDC Check

适合短视频、图文、红人内容。

核心问题：

```text
What do you carry when it gets cold?
```

内容方向：

- `What's in my winter bag`
- `Winter EDC check`
- `Campus carry check`
- `Tiny things I carry when it gets cold`
- `Under-$15 cold-weather EDC`

### 2. Cold Days. Good Vibes.

适合 meme、sticker、doodle、街头感内容。

内容方向：

- 等车五分钟像半小时。
- 冷到手指不想打字。
- 图书馆 / 办公室太冷。
- 早八上课路上的冷。
- 评论区选择题：`Which cold moment are you?`

### 3. Under-$15 Gift Finds

适合礼品季、联盟营销、Amazon finds。

内容方向：

- `Under-$15 gifts people actually use`
- `Stocking stuffers that are not random`
- `Secret Santa gift under $15`
- `Winter EDC gifts for students`
- `One for your bag, one for your desk, one for a friend`

### 4. Catch the Warmth

适合作为 Diveblues 从 `Catch the Vibe` 延展出的冬季主线。

内容方向：

- `Catch the breeze in summer. Catch the warmth in winter.`
- `Carry the Vibe. Catch the Warmth.`
- `Cold days, same vibe.`

## 红人 / 联盟可用角度

### 红人方向

- EDC / pocket gear creators：把 Go 放进 daily carry。
- Campus / dorm creators：早八、宿舍、图书馆、上课路上。
- Budget gift creators：under-$15 gift、stocking stuffer、Secret Santa。
- Commuter creators：地铁、公交、咖啡排队、办公室。
- Meme / relatable skit creators：冷天小崩溃、手冷段子、选择题。

红人 brief 的核心不是讲参数，而是：

```text
Show where it fits in your day.
```

### 联盟方向

- `under-$15 hand warmer`
- `under-$15 winter gifts`
- `stocking stuffers under $15`
- `winter EDC essentials`
- `cold-weather everyday carry`
- `campus winter essentials`
- `commuter winter essentials`
- `small gifts people actually use`

## 与 Ocoopa x Diveblues 冬季双品牌方案的关系

这份视觉策略应服务于当前双品牌分工：

- Ocoopa：Sense / S6、老品、O 品牌资产、旗舰温暖、`Nice to Hold You #2`。
- Diveblues：Go、under-$15、EDC、seasonal carry、年轻化、礼品季低门槛入口。

它补强的是 Diveblues 这侧，尤其是：

- 为什么 Diveblues Go 不能做成 Ocoopa 的低价版。
- 为什么 Diveblues Go 需要年轻、时尚、好玩、明快。
- 为什么包装和视觉要快速建立记忆点。
- 为什么温暖表达不必困在橙红暖色里。
- 为什么 EDC / seasonal carry 是更适合 Diveblues Go 的表达框架。
