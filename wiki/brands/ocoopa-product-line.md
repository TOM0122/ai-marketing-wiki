## Product Category

Rechargeable hand warmers.

The category extends beyond "warming hands" into:

- portable warmth
- emergency phone charging
- light outdoor comfort
- thermal relief / hot compress
- gifting
- EDC / carry object
- soft wellness and body-state care

## Product Family Logic

The decks divide the product line roughly into:

### H Series

Position:

```text
single-unit, practical, high value, broad daily use
```

Typical role:

- basic warmth
- power bank function
- light outdoor and daily carry
- high cost performance

Example products:

- HotPal / 118s
- HotPal PD
- H01
- H01 PD
- H01 Pro
- HeatCube

### U / Union Series

Position:

```text
modular, paired, detachable, more playful warmth experience
```

Typical role:

- two-piece magnetic use
- sharing
- larger heat surface
- changeable shells / batteries
- more scenario-specific design

Example products:

- UT2S
- UT2S mini
- UT3 Lite
- UT3 Pro
- U5
- U5S
- UT4 Urban
- UT4 Extreme
- UT4 Rare
- UT4 Young
- UT2S Gamerpro

## Product Segments

### Basic Warmth / Value

Products that should be explained as simple, useful, easy to operate, and cost-effective.

Useful scenes:

- walking
- commuting
- home use
- light outdoor
- pet walking
- gifting

Marketing angle:

```text
the reliable first hand warmer
```

### Fast-Charge / Power Bank Warmth

Products that combine warmth with charging.

Useful scenes:

- outdoor use
- travel
- golf
- camping
- phone backup power

Marketing angle:

```text
warmth plus backup energy
```

### Outdoor / Rugged

Products with water / dust / shock resistance or extreme heating mode.

Useful scenes:

- hunting
- ice fishing
- camping
- hiking
- outdoor workers
- extreme cold

Marketing angle:

```text
warmth that can survive the environment
```

### Magnetic / Paired Warmth

Products sold as a pair or with magnetic combination.

Useful scenes:

- both hands warm
- sharing with another person
- flexible carry
- gift set
- hot compress

Marketing angle:

```text
one product, two hands, more ways to hold warmth
```

### Design / Gift / Lifestyle

Products where shape, color, print, art collaboration, or jewelry-like form matters.

Useful scenes:

- gifts for women
- lifestyle carry
- holiday gifts
- visual posting
- artistic editions

Marketing angle:

```text
warmth that looks like something you chose
```

### Gamer / RGB / Indoor Performance

Products with RGB lighting, fast charging, and game-setting cues.

Useful scenes:

- indoor gaming
- desk setup
- long sessions
- tech gift

Marketing angle:

```text
keep your hands warm and your setup alive
```

## Reusable Messaging Angles

### Function

- multi-level heating
- temperature memory
- Type-C charging
- power bank support on selected models
- fast charge on selected models
- single / double-sided heating on selected models
- IP protection on selected models

### Form

- rounded grip
- silicone anti-slip ring
- oxidized sandblasted aluminum
- soft-touch coating
- lanyard / hook position
- compact size
- visual patterns and art editions

### Feeling

- does not need to be gripped tightly
- pocketable comfort
- small but reliable warmth
- warmth as care
- gift as practical concern

## Marketing Judgment

The product line is too broad to communicate through one generic "hand warmer" message.

Better structure:

```text
product segment -> use scene -> user type -> content format
```

This makes social content easier to map:

- outdoor models -> proof / test / rugged scenes
- lifestyle models -> gift / EDC / aesthetic carry
- paired models -> sharing / couple / family / both hands
- fast-charge models -> travel / phone anxiety / utility proof
- gamer model -> desk setup / RGB / long-session comfort
