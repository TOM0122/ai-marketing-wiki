# AI + Marketing Operating System

## Definition

A personal system that uses AI to support brand strategy, planning, content, consumer insight, competitive analysis, and execution.

## Current Principle

Build the knowledge base before building agents.

Reason: agents need stable inputs, labels, judgment rules, and output formats. Without those, automation only produces more unstructured material.

## Layering

1. Source layer: original reports, articles, screenshots, campaign pages, data exports.
2. Wiki layer: distilled facts, patterns, examples, and linked concepts.
3. Judgment layer: the user's own marketing calls and reusable heuristics.
4. Workflow layer: repeatable processes such as competitor monitoring and content brief generation.
5. Agent layer: scheduled collection, classification, first-pass analysis, and report drafting.

## Near-Term Focus

Start with one workflow:

Competitor monitoring -> content teardown -> topic and brief generation.
