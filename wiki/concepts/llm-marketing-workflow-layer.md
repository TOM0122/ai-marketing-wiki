## Core Idea

LLMs should be treated as a workflow layer, not as a single writing tool.

The useful shift is:

```text
one-off prompt -> repeatable marketing workflow
```

## Practical Marketing Uses

- content ideation
- first drafts
- email variations
- ad copy variants
- SEO brief generation
- report summarization
- audience segmentation support
- chatbot / conversational interfaces
- personalized landing pages

## Operating Principle

LLMs can handle drafting and variation, but they do not replace marketing judgment.

The human still owns:

- audience understanding
- positioning
- goals
- facts
- brand voice
- final QA

## Risks

### Hallucination

LLMs may invent facts, claims, examples, or competitive advantages.

Marketing risk:

- false product claims
- weak sales collateral
- legal / reputation exposure

### Generic Brand Voice

Without a voice system, output drifts toward neutral corporate language.

Mitigation:

- brand voice guide
- examples
- do / don't list
- reusable prompt library
- review checkpoints

### Workflow Confusion

Using LLMs randomly creates scattered outputs.

The better approach is:

```text
task -> input format -> prompt -> output format -> review rule -> storage
```
