## Core Idea

TikTok is not only a media channel. It is a cultural layer.

For brand accounts, the job is not just posting product content. The job is to build:

1. persona
2. storyline
3. value output

## Three Account Tasks

### Build A Persona

The account needs a recognizable personality.

For Ocoopa, the proposed persona is:

```text
the friend who knows how to live comfortably
```

This means:

- casual, not corporate
- useful, not salesy
- prepared for daily discomfort
- comfortable with small routines

### Create A Storyline

Fragmented TikTok content still needs continuity.

Series logic gives users a reason to follow:

- repeated scenarios
- repeated host roles
- recurring tests
- comment-response formats
- problem-solution mini arcs

### Bring Value

Value can be:

- emotional recognition
- entertainment
- practical tips
- community participation
- product proof

## Benchmark Patterns

### Frost Buddy

Pattern:

```text
real user question -> product demonstration -> objection removed
```

Useful for:

- simple USP demonstration
- short-form product testing
- comment-led content

### Oura Ring

Pattern:

```text
real health concern -> metric / signal -> interpretation or action
```

Useful for:

- soft wellness framing
- body-state interpretation
- trust through data

### Away

Pattern:

```text
employee / user POV -> product embedded in real use -> trust through ordinary detail
```

Useful for:

- brand humanization
- team / behind-the-scenes content
- "not perfect, therefore believable" content

## Ocoopa Adapted Content System

### Hook

Start with underestimated cold or overlooked discomfort.

### Story

Show how the user moves from discomfort to control.

### Value

Leave the user with something beyond a product feature:

- "I have had that moment"
- "this solves a small but real problem"
- "this could be part of my routine"
- "this brand understands the scene"
