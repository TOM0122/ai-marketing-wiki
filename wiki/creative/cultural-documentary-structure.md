## Use Case

Turn historical or cultural material into watchable content by combining:

- character perspective
- cultural object
- emotional hook
- knowledge explanation
- modern relevance

## Base Structure

```text
modern character problem
-> cultural object encounter
-> time / context transition
-> dialogue with historical guide
-> skill / ritual / meaning discovery
-> return to modern life
-> applied cultural reinterpretation
```

## Three-Episode Example Pattern

### Episode 1: Food

Theme:

```text
ingredients and cooking as cultural renewal
```

Modern character:

- Chinese-style restaurant owner

Historical guide:

- Han dynasty kitchen / chef figure

Output:

- modern reinterpretation of Han food

### Episode 2: Vessels

Theme:

```text
craft preservation and reinvention
```

Modern character:

- young intangible cultural heritage inheritor

Historical guide:

- lacquerware craft master

Output:

- modern lacquerware design / short video spread

### Episode 3: Ritual

Theme:

```text
hospitality and cross-cultural exchange
```

Modern character:

- Japanese university student

Historical guide:

- Xin Zhui

Output:

- cultural research / cross-cultural bridge

## Scene-Level Device

The storyboard uses first-person perspective to create immersion.

Useful devices:

- museum object as portal
- tactile trigger
- historical sound replacing modern sound
- historical guide mistaking the visitor as a participant
- concrete craft action before abstract cultural explanation
- return-to-modern scene where insight becomes a new work

## Reusable Content Method

Do not explain culture only as information.

Make the audience follow:

```text
person -> object -> action -> meaning -> modern use
```

This is stronger than:

```text
background -> facts -> expert explanation
```
