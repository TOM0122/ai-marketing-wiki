# 对标品牌：生活方式 DTC 怎么做内容 / EDM

章程点名的对标品牌（Rhode、Stanley、Glossier、Rare Beauty、Poppi、Sol de Janeiro、Owala）。**Glossier 在"邮件"层面最有一手参考;其余多为整体内容/品牌打法(邮件层面一手资料较少,已标注)。**

## Glossier —— 邮件层面最值得抄的范式

- **视觉**：极简、留白、柔和/粉彩色系;产品用白底微距特写,色彩"跳"出来。视觉系统高度一致,强化品牌识别与情感联结。
- **内容逻辑**：客户中心 + 社区驱动。大量 **UGC、好评、真实使用**,营造归属感;**不打硬广**,偏故事/教程/personal updates。
- **文案**：轻松、playful、带 emoji,用"社媒口语"而非营销腔;移动端大字大图;CTA 用一致的蓝色按钮 "Shop Now",干净不啰嗦。
- **新品发布**：先用 teaser + playful 文案预热 → 正式邮件突出大图 + 简短描述 + 明确 CTA → 配 mini 系列讲卖点 + UGC 展示真实使用。

> 这套几乎是章程"文案从'品牌讲产品'→'用户分享体验'、弱化卖货感、Ins 风/杂志化"的现成模板。

## 其余对标（整体打法,迁移到 EDM 内容方向）

- **Stanley**：色彩 drop(配色限定)、问题-解决短片、生活方式时刻;UGC + ASMR 冰块声做病毒内容。→ EDM 可借"配色限定 + 场景化生活时刻"。
- **Owala**：功能 + 颜值 + 可达性;UGC 主导(大量增量声量来自从未合作过的创作者)、color drops。→ EDM 借"真实用户场景 + 颜色上新"。
- **Poppi**：品牌先于功能(brand before benefits),用风格/玩味/人格去掉"健康饮料"的刻板;在 GRWM/haul 里"自然饱和式"出现。→ EDM 借"先立人格、产品自然融入生活内容"。
- **Sol de Janeiro**：aspirational 生活方式叙事 + 巴西文化、感官氛围。→ EDM 借"情绪/氛围/身份认同"叙事。
- **Rhode**：以极简米色/克制美学著称,但本轮**未找到邮件层面一手拆解 → to-source**(下次专门查 Rhode newsletter teardown)。
- **Rare Beauty**：本轮未单独检索 → to-source。

## 共同启示（→ 直接喂给 EDM 专项）

1. **编辑化 > 卖货化**：赢家的邮件像内容/杂志,不像商城 banner。
2. **UGC + 社区 + 真实场景**是质感来源,而非更多产品图。
3. **统一极简视觉系统** + **社媒口语文案** = 品牌识别与情感联结。
4. **情绪/身份认同叙事** > 卖点罗列。

## 一句话

对标品牌的共性:把邮件做成"用户视角的生活方式内容"(Glossier 是邮件层面最佳范式),而不是电商促销 banner。
