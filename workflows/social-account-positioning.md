# Workflow: Social Account Positioning

## Source Pattern

Derived from Ocoopa IG / TikTok account positioning files.

## Goal

Turn a brand and product line into platform-specific social account positioning.

The output should answer:

```text
why should this account exist, who should follow it, and what repeatable content system will make it recognizable?
```

## Inputs

- brand role
- product line / hero products
- audience segments
- platform
- benchmark accounts
- use scenarios
- tone constraints

## Step 1: Define The Account Job

Do not start with post ideas.

First define what the account must do.

Common jobs:

- build brand persona
- explain product utility
- create desirability
- generate UGC
- build trust
- support conversion
- support community interaction

Example:

```text
TikTok job = build persona + create storyline + output useful value
Instagram job = establish visual / lifestyle desirability + product context
```

## Step 2: Reframe The Product

Move from category label to user meaning.

Template:

```text
from <category object> -> <role in user life>
```

Example:

```text
from hand warmer -> tactile comfort object / winter EDC / soft wellness accessory
```

## Step 3: Identify Unspoken User Context

Find the moment users recognize but rarely say clearly.

Useful prompts:

- When does the problem appear before the user names it?
- What small discomfort is usually ignored?
- What scene makes the product feel obvious?
- What is the body doing before the user explains it?

Example:

```text
The user does not say "I need a hand warmer"; the hand notices cold first.
```

## Step 4: Build Content Pillars

Use 3-5 pillars.

Each pillar should contain:

- scene
- user tension
- product role
- content format
- value to audience

Example pillars:

- problem-driven warmth
- value reframing
- audience-based desirability
- product proof / test
- community response

## Step 5: Build Platform-Specific System

### TikTok

Use:

```text
persona -> storyline -> value output
```

Content forms:

- comment response
- product test
- POV
- host-led demo
- street reaction
- challenge
- behind the scenes

### Instagram

Use:

```text
visual desirability -> lifestyle scene -> product identity
```

Content forms:

- aesthetic use scenes
- product close-up
- EDC / bag contents
- gifting moments
- travel / outdoor scenes
- carousel education

## Step 6: Benchmark Correctly

Do not copy benchmark aesthetics only.

Extract the content mechanism.

Benchmark template:

```text
Account:
What they start from:
What tension they use:
How product appears:
Why audience stays:
What can be adapted:
```

Example mechanisms:

- real user question -> product demo -> objection removed
- health concern -> data signal -> action advice
- employee POV -> real use -> trust

## Step 7: Define Voice Boundaries

Write both yes and no.

Example:

Yes:

- personal
- useful
- tactile
- situational
- lightly warm

No:

- hard sell
- generic gadget language
- feature dump
- seasonal-only framing
- over-polished brand voice

## Output Template

```md
# <Brand> <Platform> Account Positioning

## Account Job

## Product Reframe

## Audience

## Core Context

## Content Pillars

## Recurring Formats

## Benchmark Mechanisms

## Voice

## Do Not Do

## First 10 Post Ideas
```
